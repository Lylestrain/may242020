Holiday lights

Start with an empty "div id=lights", inside this put a "div class=bulb" and in the css give it some size, width, height, margin, the background-color and box-shadow are added here for visualisation and will be applied with java-script later to make the flashing effect.

The "div" element is a generic element for styling, which is why it will work well in this application.

the second "div box" has a box shadow applied to one side,the third "div box" has a box shadow applied on both sides.

A border-radius:100% 4px makes a nice bulb shape

A transform: rotate(-45deg) straightens the bulb shape

in the css(styles) go about setting the styles(+opacity) for every other bulb:-- .bulb:nth-child(odd) --, remove background-color, box-shadow from the class bulb(.bulb)

Set the keyframes for "bulb-pulse-1" changing the opacity from 1 to 0.3.

apply the animation property to the .bulb:nth-child(odd)
